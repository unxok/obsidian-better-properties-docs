#### Arguments

| Name    | Type                                                                              | Default     |
| ------- | --------------------------------------------------------------------------------- | ----------- |
| options | <InfoButton label={<code>{"object \| undefined"}</code>} content={<Options />} /> | `undefined` |

#### Returns

[CachedMetadata](https://docs.obsidian.md/Reference/TypeScript+API/CachedMetadata)

#### Examples

```js
api.getMetadata();

api.getMetadata({ path: "path/to/note.md" });

api.getMetadata({ subscribe: false });

api.getMetadata({
  path: "path/to/note.md",
  subscribe: false,
});
```
