## Introduction

Better Properties is an [Obsidian](https://obsidian.md) plugin which adds numerous features related to [Properties](https://help.obsidian.md/properties).

<br />

<Alert>
  <TriangleAlert />
  <AlertTitle>Obsidian internal API usage</AlertTitle>
  <AlertDescription>

    changes that break this plugin after a new Obsidian update is released.
    Please [open an
    issue](https://github.com/unxok/obsidian-better-properties/issues) and/or be
    patient in the event that this happens.
  </AlertDescription>
</Alert>
